Solve using factoring:

x2+8x+12

  1. Find all the factors(x+2 and x+6)

  2. To find the x intercepts, set each factor to (factor)=0

  3. So, it would be x+2=0 and x+6=0, which are x=-2 and x=-6

  4. So the graph vertex(peak) is on the left of the y-axis

  5. The point in the middle is x=-4, so plug -4 in for x.

  6. -42+8*(-4) +12 = -4

  7. This means the graph is opening upwards.
