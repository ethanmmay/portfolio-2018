My Action Pictures How I Made This Picture

During this project we learned how to use Graphic Image Manipulation Tool (GIMP) to create layers and mask them in order to copy the same object multiple times throughout a picture. In this example it was my friend Colter doing a backflip in front of Cenntennial Highschool

We then exported the image as a GIF and animated it as well as picked the speed of the animation.
