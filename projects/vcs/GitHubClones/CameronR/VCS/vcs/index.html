VCS

VCS, or Version Control System, is used by almost everyone that writes code. It is very helpful because it allows you to share code with other members of your team very easily. It also allows you to easily merge your code with other poeple's code.

Git

Git is one example of this system. It is very easy to use, and it simplifies the process of sharing your code even more. If you are reading this, tell me. It is also the first version of VCS that was created.

Github

GitHub is what Git uploads the files to. It is a workspace where everyone can share files and code with everyone else. Like Git, it is easy to use, and it makes it much easier to share code with groups of people.

Command line code for GIT on linux to upload files to Github.

Setting up GIT, one time use code.

compsci@compsci-VBox:/var/www/html/cameronr/projects$ git user.name 'name'
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git user.email 'email'
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git config

Getting GIT ready for upload, then uploading.

compsci@compsci-VBox:/var/www/html/cameronr/projects$ git init
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git add -A (or any file name)
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git status
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git commit -m 'Commit Description'
        
compsci@compsci-VBox:/var/www/html/cameronr/projects$ git log

Retrieving code with GIT

compsci@compsci-VBox:/var/www/html/cameronr/projects$ git clone (url)
